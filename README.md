## Quick Start

Clone the repository

Install dependencies, then run the development server:

```bash
yarn
yarn dev
```

# Deploy

Deploy to the cloud with [Vercel](https://vercel.com/)

Install the `now` CLI

```bash
npm install --global vc
```

Then deploy using

```bash
vc
```
